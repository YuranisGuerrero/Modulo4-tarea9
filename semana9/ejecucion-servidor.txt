Cuando ejecuto el comando node server.js, el servidor inicia y queda 
esperando solicitudes de los usuarios.

En la terminal aparece el mensaje: "Servidor ejecutándose en puerto 3000", 
lo que indica que el servidor está funcionando correctamente.

Al visitar la ruta http://localhost:3000 se muestra el mensaje "Servidor 
activo y funcionando".

Cuando ingreso a la ruta http://localhost:3000/estado, el servidor responde 
con un objeto JSON que informa que el servicio está funcionando y que 
corresponde a una API comunitaria.
